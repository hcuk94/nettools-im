Special Bits

Locally Administered Bit (Bit 1)

When set, indicates the MAC address was assigned locally (not by manufacturer). Common in virtual machines, containers, and privacy features.

x2:xx:xx:xx:xx:xx, x6:xx:xx:xx:xx:xx, xA:xx:xx:xx:xx:xx, xE:xx:xx:xx:xx:xx

Multicast Bit (Bit 0)

When set, indicates a multicast address (sent to multiple devices). Unicast addresses have this bit cleared.

x1:xx:xx:xx:xx:xx, x3:xx:xx:xx:xx:xx, x5:xx:xx:xx:xx:xx, ...

IEEE Registration Types

Type Prefix Size Address Block Description
MA-L 24-bit (6 hex) ~16.7M addresses Large block for major vendors (most common)
MA-M 28-bit (7 hex) ~1M addresses Medium block for mid-size manufacturers
MA-S 36-bit (9 hex) 4096 addresses Small block for limited production
IAB 36-bit (9 hex) 4096 addresses Individual Address Block (legacy, replaced by MA-S)
